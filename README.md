# Prodigy InfoTech Web Development Internship - Task 2: Stopwatch Web Application

A premium, interactive stopwatch web application built using HTML5, CSS3 (Glassmorphism UI), and Vanilla JavaScript. 

## Features
- **Accurate Time Tracking:** Measures elapsed time down to milliseconds using `Date.now()`.
- **Lap Tracking System:** Dynamically logs and tracks lap historical checkpoints.
- **Glassmorphic UI Design:** Modern dark theme with ambient lighting effects and neon accents.
- **State Management:** Intuitively handles button states (disabled/enabled) to prevent execution errors.

## Technologies Used
- HTML5 (Structure)
- CSS3 (Custom Scrollbars, Keyframe Animations, Glassmorphism Backdrop-filter)
- JavaScript (DOM Manipulation, Interactivity, Timing Events)
